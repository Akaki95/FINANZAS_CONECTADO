Este proyecto de aplicación de finanzas personales no tendrá backend.
Toda la lógica de negocio, validaciones, cálculos y visualizaciones se ejecutan directamente en el frontend (HTML/JS/CSS).
La única conexión externa será MongoDB Atlas para almacenar y sincronizar datos.
No se deben generar endpoints REST ni GraphQL.
Cualquier procesamiento, filtrado, agregación o dashboard se hará en el frontend.
En un futuro, si se agrega multiusuario, cada usuario configurará su propia conexión Atlas; no habrá servidor centralizado.
